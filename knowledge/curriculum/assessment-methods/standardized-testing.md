# Standardized Testing

Uniform assessment administered to all students under same conditions.

## Types
- State accountability assessments
- District benchmarks
- Nationally normed tests (Iowa, Stanford)
- College entrance exams (SAT, ACT)
- English proficiency assessments (ACCESS)

## Purpose
- Comparability across students/schools
- Accountability and program evaluation
