# Common Core State Standards (CCSS)

## English Language Arts
- **Key Shifts:** Content-rich nonfiction, evidence-based reading/writing, complex text
- **Strands:** Reading, Writing, Speaking/Listening, Language
- **Literacy:** History/Science technical subjects included

## Mathematics
- **Key Shifts:** Focus, coherence, rigor (conceptual understanding, procedural skill, application)
- **Domains:** Operations, algebra, geometry, measurement, data, fractions, ratios, functions, statistics

Focus on college and career readiness
