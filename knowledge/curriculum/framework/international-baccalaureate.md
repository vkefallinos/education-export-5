# International Baccalaureate (IB)

## Programmes
- **PYP (ages 3-12):** Transdisciplinary, learner profile focus, exhibition
- **MYP (ages 11-16):** Subject groups, global contexts, personal project
- **DP (ages 16-19):** Diploma programme, university recognition

## Key Features
- Concept-based curriculum
- International-mindedness
- Approaches to teaching and learning skills
