# Hybrid Learning

## Characteristics
- Some students in-person, some remote (or alternating days)
- Synchronous + asynchronous elements
- Technology connects all learners

## Best For
- Concurrent teaching strategies
- Station rotation models
- Digital collaboration tools
- Flexible learning environments

## Challenge
- Teacher attention divided between settings
