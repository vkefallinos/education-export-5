# Interactive Whiteboard

## Uses
- Modeling think-alouds and problem-solving
- Virtual manipulatives and tools
- Student demonstrations and sharing
- Interactive games and activities
- Annotation of texts and diagrams

## Keep Active
- Avoid passive viewing
- Have students interact with the board
- Always backup plan for technical issues
