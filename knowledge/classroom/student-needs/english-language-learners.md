# English Language Learners

## Proficiency Levels
Entering → Beginning → Developing → Expanding → Bridging

## Key Strategies
- **Visual Supports:** Images, gestures, graphic organizers, realia
- **Scaffolding:** Sentence frames, word banks, modeled language
- **Comprehensible Input:** Clear speech, written notes, avoid idioms
- **Wait Time:** Extra processing time for language formulation

## Assessment
- Separate content knowledge from English proficiency
- Allow multiple demonstration formats
- Extended time for assessments
